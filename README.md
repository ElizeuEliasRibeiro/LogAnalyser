# Log Analyzer

## Objetivo

Projeto em Python para analisar logs SSH do Linux e detectar possíveis ataques de força bruta.

## Funcionalidades

- Leitura de auth.log
- Contagem de tentativas por IP
- Identificação de IPs suspeitos
- Geração de relatório
- Estatísticas da análise

## Tecnologias

- Python
- Linux
- SSH
- Git

## Como executar

python src/main.py

## Exemplo de saída

IP: 192.168.1.10
Tentativas: 8
Status: SUSPEITO
